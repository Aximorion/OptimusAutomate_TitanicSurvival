# OptimusAutomate_TitanicSurvival

## Project Overview
This project is part of the Machine Learning Virtual Internship at Optimus Automate. The objective was to build and evaluate a classification model to predict passenger survival on the Titanic.

## Steps Performed
- **Exploratory Data Analysis (EDA):** Analyzed survival distribution based on passenger demographics.
- **Data Preprocessing:** Handled missing age values and encoded categorical variables for model readiness.
- **Model Training:** Implemented and compared two classification models:
  - Logistic Regression
  - Random Forest
- **Evaluation:** Compared models based on Accuracy, Precision, Recall, and F1-Score.

## Results
- **Logistic Regression Accuracy:** 0.81
- **Random Forest Accuracy:** 0.81

## Technologies Used
- Python
- Pandas, Seaborn, Matplotlib
